# Tradutor Tupife

Tradutor e transpositor da **notação numérica Tupife** para pífanos.

O sistema Tupife é uma tablatura de furos: números de 0 a 6 indicam a digitação,
`)` indica forquilha (meio furo), `'` a 2ª oitava, `''` a 3ª oitava e `~` a
sustentação da nota.

- **Digitações → Notas:** digite a sequência de um pife (Dó, Sol ou Ré) e veja as
  notas que soam, o tom provável e a transposição automática para os outros pífanos.
- **Notas → Digitações:** digite as notas (dó ré mi… ou c d e f g a b) e veja a
  digitação nos três pífanos.

Feito para a **Pifarada** — orquestra de pífanos do Rio de Janeiro.

---

## Como usar

É um site de **arquivo único**. Não precisa instalar nada, não precisa de build.

- **Testar no computador:** dê dois cliques em `index.html` — abre no navegador.
- **Publicar de graça no GitHub Pages:**
  1. Crie um repositório no GitHub e suba o `index.html`.
  2. No repositório, vá em **Settings → Pages**.
  3. Em **Source**, escolha a branch `main` e a pasta `/ (root)`.
  4. Salve. Em um minuto o site fica no ar em
     `https://SEU_USUARIO.github.io/NOME_DO_REPO/`.

Também funciona em qualquer hospedagem de site estático (Vercel, Netlify,
Cloudflare Pages): é só apontar para o `index.html`.
